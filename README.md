# Discord.SimpleTransparency
A super simple transparent theme for discord.

## Installation
Download the file and add it to your BetterDiscord or Vencord themes folder.
Or, if using Vencord, paste the following link into the "online themes" area in the themes tab:
`https://raw.githubusercontent.com/Ultragamer135/Discord.SimpleTransparency/refs/heads/main/SimpleTransparent.theme.css`
